# WMC-Jakob: Materialien zur Web- und Multimedia-Content Entwicklung

Dieses Repository sammelt Übungen und Codebeispiele, die im Rahmen des Kurses zur Web- und Multimedia-Content Entwicklung (WMC) entstanden sind. Der Fokus liegt auf den Grundlagen von **HTML** und **CSS**, sowie modernen Layout-Techniken und Frameworks.

## Repository-Ordnung

Die Ordnerstruktur ist nach den behandelten Web-Themen organisiert:

| Ordner | Beschreibung | Technologien |
| :--- | :--- | :--- |
| **`Bootstrap`** | Beispiele und Übungen zum CSS-Framework **Bootstrap**. | HTML, CSS, Bootstrap |
| **`CSS_Diner`** | Übungen zu **CSS Selectors** (angelehnt an CSS Diner). | HTML, CSS |
| **`CSS_Garden`** | Übungen zu **CSS Layout** und **Eigenschaften** (angelehnt an CSS Zen Garden). | HTML, CSS |
| **`FlexFrog`** | Spezifische Übungen zum Verständnis und zur Anwendung von **Flexbox** (angelehnt an Flexbox Froggy). | HTML, CSS (Flexbox) |
| **`Grid`** | Übungen zum **CSS Grid Layout** System. | HTML, CSS (Grid) |
| **`Testing`** | Ordner für allgemeine Tests oder temporäre Code-Snippets. | HTML, CSS |

## Hauptdateien

* **`index.html`**: Die Haupt-HTML-Datei für die primären Beispiele.
* **`styles.css`**: Die zugehörige CSS-Datei.

## Verwendete Sprachen

* HTML
* CSS
